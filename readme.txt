== Installation ==

= The easy way: =

1. Go to the Plugins Menu in WordPress
1. Search for "WooCommerce PayU India Payment Gateway"
1. Click 'Install'
1. Activate the plugin

= Manual Installation =

1. Download the plugin file from this page and unzip the contents
1. Upload the `payu-india-payment-gateway-woocommerce` folder to the `/wp-content/plugins/` directory
1. Activate the `payu-india-payment-gateway-woocommerce` plugin through the 'Plugins' menu in WordPress
